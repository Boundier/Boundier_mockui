# Boundier Demo (Updated)

A concept demo showcasing how the expanded Conscient Technology engine analyzes psychological influence, reality distortion, and echo‑chamber drift inside social media content.

## Overview

This demo illustrates the upgraded Conscient engine in action. It analyzes:

1. **Influence signals** (fear, urgency, hype, authority, etc.)
2. **Distortion patterns** (emotional overload, selective framing, narrative skew)
3. **Exposure drift** (echo‑chamber risk and feed diversity)
4. **User reactions** (engagement, hesitation, fixation)

Everything updates in real time to reflect how your information environment affects you.

**Tech Stack:**

* React + Vite
* Tailwind CSS
* Framer Motion
* Recharts
* LocalStorage (for persistence)

---

## Setup & Run

1. Install dependencies:

   ```bash
   npm install
   ```
2. Start the development server:

   ```bash
   npm run dev
   ```

---

## Scoring Rules (Conscient Engine)

The engine uses deterministic rule-based logic to compute Influence Vectors, Response Vectors, Distortion Signals, and Exposure Indicators.

### **Influence Vector**

Calculated from content features:

* **Fear:** "warn", "danger", "risk", threatening imagery.
* **Urgency:** "now", "today", countdown cues, intense punctuation.
* **Hype:** clickbait phrases, exaggerated claims.
* **Authority:** experts, studies, official references.
* **Curiosity:** open loops, unresolved questions.
* **Visual Hype:** saturated thumbnails, expressive faces.

### **Distortion Signals**

Checks for structural reality-bending cues:

* **Emotional Overload:** high emotional intensity used to overshadow logic.
* **Selective Framing:** one-sided evidence, cropped context.
* **Narrative Skew:** biased storyline direction or claim shaping.
* **Claim-Source Mismatch:** big claims backed by weak or unclear sources.

### **Exposure Indicators**

Tracks long-term information environment health:

* **Echo Risk:** repeated exposure to the same viewpoint or narrative.
* **Viewpoint Spread:** variety of perspective signals.
* **Repetition Rate:** how often identical angles or claims recur.
* **Source Variety:** range of unique content sources.

### **Response Vector**

Calculated from user interaction patterns:

* **Engagement:** dwell time.
* **Hesitation:** rapid open–exit behavior.
* **Fixation:** repeated revisits or long stare time.
* **Clickbait Response:** how quickly the user taps after provocative cues.

---

## Dashboard Features

The demo visualizes:

* **Vulnerability Radar:** tracks susceptibility to influence cues.
* **Distortion Bars:** shows exposure to warped narratives.
* **Echo & Exposure Cards:** highlight echo-chamber risk and feed diversity.
* **Integrated Pattern Graph:** overlays influence (blue), distortion (red), and echo risk (yellow) over time.

---

## Data & Privacy

This is a client-side only demo. All analysis happens inside your browser, and all data is stored locally via LocalStorage.

**No data ever leaves your device.**

### Resetting Data

Use the trash icon in the Dashboard to wipe all stored data.
